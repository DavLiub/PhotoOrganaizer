# Product Roadmap

## Release 1.0

Reliable Android photo backup MVP:

- local photo access;
- Google Drive connection;
- local index;
- optimized backup upload;
- status tracking;
- Free/Premium boundaries.

## Later Releases

Future releases may add:

- restore;
- search;
- video support;
- multiple cloud providers;
- duplicate and similar-photo workflows;
- AI-assisted organization;
- cleanup flows;
- richer statistics.

## Planning Rule

Do not include future-release functionality in Release 1.0 implementation unless an explicit PR plan changes the scope.
